1. Python क्या है?

Python एक high-level, interpreted programming language है जो आसान syntax और powerful libraries के लिए जानी जाती है। यह beginner-friendly होने के साथ-साथ professional development के लिए भी उपयोगी है।

Python की खासियतें:

  • Simple और readable syntax
  • Open-source और free
  • Cross-platform support (Windows, Mac, Linux)
  • Rich standard library
  • Dynamic typing
  • Large community और बहुत सारे frameworks

7. File Handling (फाइल प्रबंधन)

Python में फाइल पढ़ने और लिखने के लिए open(), read(), write() functions का उपयोग होता है।


# फाइल में लिखना
with open("data.txt", "w") as file:
    file.write("यह एक लाइन है।")

# फाइल पढ़ना
with open("data.txt", "r") as file:
    content = file.read()
    print(content)

8. Exception Handling (त्रुटि प्रबंधन)

Errors को handle करने के लिए try-except block का उपयोग करें।

12. Advanced Topics

12.1 Generators

Generators lazy evaluation करते हैं और memory बचाते हैं।


def gen():
    yield 1
    yield 2
    yield 3

for i in gen():
    print(i)

12.2 Decorators

Functions को modify करने के लिए decorators का उपयोग।


def decorator_func(func):
    def wrapper():
        print("Function से पहले का काम")
        func()
        print("Function के बाद का काम")
    return wrapper

@decorator_func
def say_hello():
    print("नमस्ते!")

say_hello()

12.3 Lambda Functions


square = lambda x: x * x
print(square(5))  # 25

12.4 List Comprehensions


squares = [x * x for x in range(5)]
print(squares)  # [0, 1, 4, 9, 16]
